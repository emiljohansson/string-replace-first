# string-replace-first

> Replaces first match of a pattern in a string.

## Usage

```js
var replaceFirst = require('string-replace-first');
replaceFirst('path/to/file', '/', '-');
// => 'path-to/file'
```

## API

### replaceFirst(str, pattern, newSubStr)

Returns the modified string.

#### str

Type: `string`

#### pattern

Type: `string`

#### newSubStr

Type: `string`
